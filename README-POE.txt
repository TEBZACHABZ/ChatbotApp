ChatbotApp - Cybersecurity Awareness Assistant

ChatbotApp is a WPF-based desktop application built in C# that helps users stay cyber-aware with four interactive features:

-  Chatbot: A simulated NLP assistant for basic cybersecurity advice
-  Task Assistant: Set personal reminders or cybersecurity tasks
-  Quiz: Test your cybersecurity knowledge with a multiple-choice quiz
-  Activity Log: Track all interactions in one place

1. Chatbot

- Adds all user interactions to the activity log

2. Task Assistant
- Allows users to create tasks and reminders
- Inputs include a task title, description, and due date
- Tasks are listed and also logged in the activity log

3. Cybersecurity Quiz
- Multiple choice questions on password safety, phishing, and other topics
- Tracks score and gives immediate feedback
- Score and results are recorded in the log

4. Activity Log
- Logs all user interactions from other sections
- Read-only history of chatbot chats, tasks, and quiz results
